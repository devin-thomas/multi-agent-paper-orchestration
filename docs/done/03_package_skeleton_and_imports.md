# Task 03: Package Skeleton And Imports

## Goal

Move from a single-file submission to an importable Python package without changing behavior.

## Scope

- Create `src/paper_orchestration/`.
- Add modules for config, catalog, database, schemas, parsing, pricing, tools, agents, and evaluation.
- Move code in small chunks, keeping imports explicit.
- Avoid redesigning the working system during the move.

## Done When

- The package imports cleanly.
- Existing behavior has a clear new home.
- The legacy file remains available as a reference but is no longer the intended edit target.
