# Task 01: Bootstrap Repository

## Goal

Create a clean GitHub-ready repository shell that preserves the passing capstone artifacts and makes the next work session self-contained.

## Scope

- Initialize Git.
- Preserve the passing source as `legacy/project_starter_passing.py`.
- Preserve the passing runtime dependencies as `legacy/requirements_passing.txt`.
- Add input datasets under `data/`.
- Add preserved evaluation output under `examples/`.
- Add the main plan under `docs/portfolio_refactor_plan.md`.
- Add ten task briefs under `docs/tasks/`.

## Done When

- `git status` is clean after the initial commit.
- A future contributor can understand the repo direction from `README.md` and `docs/portfolio_refactor_plan.md`.
- No secrets, local databases, logs, or virtual environments are committed.
